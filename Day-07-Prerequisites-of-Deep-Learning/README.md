# Day 07 – Prerequisites of Deep Learning (Machine Learning Revision)

📅 **Date:** 25 June 2026

## 🎯 Objective

Revise the core Machine Learning concepts required before studying Deep Learning. This session focused on understanding different learning paradigms, regression, optimization algorithms, model evaluation techniques, overfitting, regularization, and the relationship between Machine Learning and Deep Learning.
