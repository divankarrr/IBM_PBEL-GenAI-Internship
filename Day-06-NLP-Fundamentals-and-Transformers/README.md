# Day 06 – Natural Language Processing (NLP) Fundamentals & Transformers

📅 **Date:** 24 June 2026

## 🎯 Objective

Learn the core concepts of Natural Language Processing (NLP), including text preprocessing techniques, feature extraction methods, Named Entity Recognition (NER), and an introduction to Transformer-based language models using the Hugging Face Transformers library.

---

## 📚 Topics Covered

### Classical NLP

- Sentence Tokenization
- Word Tokenization
- Stemming
- Lemmatization
- Stopword Removal
- Count Vectorizer
- TF-IDF
- N-Grams
- Part-of-Speech (POS) Tagging
- Named Entity Recognition (NER)

### Modern NLP

- Transformers
- Hugging Face Pipeline
- GPT-2 Text Generation
- Temperature
- Top-K Sampling
- Top-P (Nucleus) Sampling
- Multiple Text Generation
- Using Different Transformer Models
