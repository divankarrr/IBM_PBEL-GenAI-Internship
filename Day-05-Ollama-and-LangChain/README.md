# Day 05 – Local LLMs with Ollama & LangChain

📅 **Date:** 23 June 2026

## 🎯 Objective

Learn how to set up a local Large Language Model (LLM) environment using Ollama and integrate it with LangChain to perform AI-powered text generation.

---

## 📚 Topics Covered

- Introduction to Ollama
- Installing Ollama on Linux
- Running Ollama Server
- Pulling AI Models
- LangChain Integration
- Prompt Templates
- Running Local LLMs

---

## 🛠️ Tasks Performed

### 1. Installed Ollama

- Updated Linux packages
- Installed required dependencies
- Installed Ollama

### 2. Started Ollama Server

Created a background thread to start the Ollama service.

### 3. Downloaded AI Models

Explored different models including:

- Granite 4.1 (3B)
- Llama 3.2
- Gemma

### 4. Installed LangChain-Ollama

Integrated Ollama with LangChain for AI workflows.

### 5. Created Prompt Template

Designed a reusable prompt using LangChain's `ChatPromptTemplate`.

### 6. Connected LangChain with Ollama

Used `OllamaLLM` to communicate with a locally running language model.

### 7. Generated AI Responses

Executed prompts and generated answers using the Llama model.
