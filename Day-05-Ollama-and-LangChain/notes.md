# Day 05 Notes

## Ollama

Ollama allows developers to run Large Language Models locally without relying on cloud APIs.

Popular Models:

- Granite
- Llama
- Gemma
- Mistral

---

## LangChain

LangChain is a framework used to build applications powered by Large Language Models.

Main Components:

- Prompt Templates
- Chains
- Models
- Memory
- Agents

---

## Prompt Template

Prompt templates allow dynamic user input.

Example:

Question: {question}

Answer: Let's think step by step.

---

## Ollama Commands

Install Ollama

ollama serve

Pull a model

ollama pull granite4.1:3b

List installed models

ollama list

Run a model

ollama run llama3.2
